### 👋🏻 안녕하세요

소마인들의 오픈소스 공간 SW Maestro OSS 입니다.

소마 프로그램에서 좋은 예비 개발자 연수생들과 멘토들이 유용한 코드를 공유하고, 더 나아가 오픈소스 생태계에 마에스트로에서 보낸 시간을 통해 산출된 코드를 기여하기 위한 목적으로 생성하였습니다.
